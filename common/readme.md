This directory contains code to generate the analysis samples used in the **externality** and **merger** analyses.

The samples are built from a local database that integrates multiple public data sources, including:

- **Airline On-Time Performance Data (OTP)** — U.S. Department of Transportation
- **Local Climatological Data (LCD)** — NOAA / National Centers for Environmental Information
- **Releasable Aircraft Database** — U.S. Federal Aviation Administration
- **GDP by Metropolitan Area** — U.S. Bureau of Economic Analysis
- **American Community Survey (ACS)** — U.S. Census Bureau
